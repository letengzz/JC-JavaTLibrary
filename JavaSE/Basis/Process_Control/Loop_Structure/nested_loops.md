# 嵌套循环

将一个循环结构A声明在另一个循环结构B的循环体中，就构成了**嵌套循环**

```java
外层循环:循环结构B
内层循环:循环结构A
```

**说明**：

​	内层循环结构遍历一遍，只相当于外层循环体循环一次

**技巧**：

​	**外层循环控制行数，内层循环控制列数**嵌套循环
