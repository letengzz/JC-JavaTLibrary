# GsonFormatPlus

![image-20221202113952214](https://cdn.jsdelivr.net/gh/letengzz/Two-C@main/img/Java/202212031621735.png)

根据 json 生成对象。

**使用**：使用alt + s 或 alt + insert调取。

```json
{
	"name": "tom",
	"age": "18",
	"gender": "man",
	"hometown": {
		"province": "河北省",
		"city": "石家庄市",
		"county": "正定县"
	}
}
```
